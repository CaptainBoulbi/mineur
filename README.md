# MINEUR

Minesweeper game.

# building

You need to install [raylib v5](https://github.com/raysan5/raylib/releases/tag/5.0), but any version should work.

Oh no, my os doesn't comme with minesweeper preinstalled, let's make it myself.

```console
$ make
```

to run

```console
$ ./mineur
```

or

```console
$ make run
```
